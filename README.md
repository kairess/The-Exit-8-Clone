# The Exit 8

The Exit 8, game clone using Python, Ursina game engine

8번 출구 게임 만들기

## 설치법

### uv 패키지 관리자
```bash
uv pip install -r requirements.txt

# 실행
uv run main.py
```

### pip를 통한 설치
```bash
pip install ursina
```
or
```bash
pip install -r requirements.txt

# 실행
python main.py
```
